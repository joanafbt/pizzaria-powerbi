# 🍕 Análise de Vendas e Gestão de uma Pizzaria

## Objetivo
Analisar os padrões de vendas e o consumo de ingredientes de uma pizzaria,
com o objetivo de apoiar decisões estratégicas na gestão do negócio —
desde a otimização de stock até à definição de estratégias comerciais.

## Ferramentas e tecnologias
- Power BI (desenvolvimento do dashboard interativo)
- Dataset: dados de vendas e ingredientes de uma pizzaria

## O que o dashboard analisa
- Vendas por produto — identificação dos produtos mais e menos vendidos
- Receita gerada — acompanhamento do desempenho financeiro
- Padrões temporais — análise de picos de vendas (dias, semanas, meses)
- Consumo de ingredientes — monitorização do uso de matérias-primas

## Principais insights
- Determinados produtos concentram a maior parte das receitas
- Existem picos de vendas em períodos específicos da semana
- Alguns ingredientes apresentam consumo elevado e impacto direto no stock
- Produtos com baixo volume de vendas podem estar a reduzir a margem

## Aplicação prática
- Otimização de stock com base nos ingredientes mais consumidos
- Apoio à definição de promoções para produtos com menor saída
- Melhoria do planeamento operacional com base em padrões temporais
- Tomada de decisão baseada em dados para gestão do negócio

## Valor do projeto
Este projeto demonstra capacidade de:
- Criação de dashboards interativos em Power BI
- Análise de dados aplicada a um contexto de negócio real
- Extração de insights relevantes para decisões estratégicas
- Comunicação visual de informação complexa de forma clara
